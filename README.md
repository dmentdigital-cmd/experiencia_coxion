# Coxión: Cocina y Conexión (Landing Page)

Landing page premium de alta conversión diseñada para **Coxión**, un servicio de catering de autor y experiencias gastronómicas boutique en Bogotá. Desarrollado con los estándares de ingeniería de crecimiento de **Dmente Digital**.

## 🌟 Propuesta de Valor
Coxión no es solo catering; es la orquestación sensorial de encuentros. El sitio web refleja esta filosofía a través de un diseño "Editorial High-End" que prioriza la conexión humana y la sostenibilidad gastronómica.

## 🚀 Características Principales
- **Identidad:** Uso de la paleta verde salvia y beige arena para transmitir naturalidad y exclusividad.
- **Responsabilidad:** Enfoque en ingredientes agroecológicos, producción artesanal de salsas/condimentos y gestión integral de residuos (compostaje).
- **Chef-First:** Equipo conformado por **Alex** (Catering de Autor y Fermentación) y **Cami** (Pastelería y Diseño de Experiencias), elevando la confianza y el detalle visual.
- **Cero Ultraprocesados:** Compromiso total con la cocina real, eliminando conservantes y aditivos industriales en todas las preparaciones.
- **Arquitectura de Conversión:** CTAs estratégicos, botón flotante de WhatsApp y Lead Magnet ("Guía de Anfitrión").
- **Optimización Multidimensional (SEO/GEO/AEO):**
  - **SEO Técnico:** Meta tags, Open Graph y estructura semántica.
  - **GEO SEO:** Posicionamiento local optimizado para Bogotá, Colombia.
  - **AEO (IA):** Implementación de FAQPage Schema.org para visibilidad en ChatGPT, Perplexity y Gemini.
- **Branding Dmente Digital 2026:**
  - Tipografía `Plus Jakarta Sans`.
  - Lema corporativo *"No es azar, es propósito"*.
  - Atribución oficial y protocolos de seguridad.
- **Analytics & Tracking:** Integración nativa de Google Analytics 4 (GA4) y Meta Pixel.
- **Mobile-First Design:** Interfaz fluida y optimizada para dispositivos móviles (80% del tráfico esperado).

## 🛠️ Requisitos Técnicos
- **Navegador:** Cualquier navegador moderno (Chrome, Safari, Edge, Firefox).
- **Entorno:** No requiere servidor de backend (Standard HTML/CSS/JS).
- **Scripts:** Python 3.8+ (opcional, para herramientas de búsqueda en `deep_search_projects.py`).

## 💻 Instalación y Ejecución Local
1. **Clonar/Descargar** el repositorio.
2. **Ejecutar:** Simplemente abre `index.html` en tu navegador.
3. **Desarrollo:** Se recomienda usar la extensión *Live Server* de VS Code para previsualización en tiempo real.

## 🛡️ Protocolos Aplicados
- **Seguridad Antigravity:** Auditoría de fugas de datos y protección contra prompt-injection.
- **Analytics Protocol:** Configuración de eventos de conversión y seguimiento de usuarios.
- **Design System:** Implementación de Tailwind CSS con paleta personalizada "Natural Gourmet".
